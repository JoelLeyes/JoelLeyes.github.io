Topología de Red (VPC y Security Groups)

Estructura de red en AWS mostrando la VPC, subnets, Security Groups aplicados a EC2, las reglas de ingreso/egreso (puertos 80, 443, 5432 y SSH), y la exposición pública del Load Balancer de EC2.

Flujo de Despliegue (GitHub Actions + Docker + AWS)

Pipeline de CI/CD automatizado que comprende: push a GitHub → GitHub Actions dispara builds, tests y generación de imágenes Docker → push a ECR → actualización automática en EC2 con nuevo contenedor. Incluye notificaciones y rollback en caso de fallos.

Diagrama de pipeline aún pendiente de integración en esta vista

Componentes Internos (Backend)

Arquitectura en capas del backend Java EE mostrando la estructura de Servlets, Services, DAOs, modelos de dominio, integraciones de email (EmailService), push notifications, y persistencia en PostgreSQL.
