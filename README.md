## Proyectos Destacados

### 📋 [Sistema de Registro de Usuarios en AWS](index.html#proyecto-registro)
Arquitectura de microservicios en Amazon EKS con:
- **Backend:** Python + Django + PostgreSQL (RDS)
- **Microservicios:** Flask (notificaciones), Nginx (frontend)
- **Seguridad:** ALB + ACM (HTTPS), API Gateway, SSDLC (Bandit + pip-audit + OWASP ZAP)
- **Repositorios:** [Backend](https://github.com/JoelLeyes/BackEnd-proyecto_registro_usuarios) | [Frontend](https://github.com/JoelLeyes/FrontEnd-proyecto_registro_usuarios) | [Notificaciones](https://github.com/JoelLeyes/Notificaciones-proyecto_registro_usuarios)

### 📅 [AgendaOnline: Marketplace SaaS con Observabilidad](index.html#proyecto-agenda)
Plataforma de servicios profesionales en Amazon EKS con logging en MongoDB Atlas:
- **Backend:** Laravel 10 + REST API + Sanctum
- **Frontend:** Vue 3 + Vite + Pinia + PWA
- **Observabilidad:** MongoDB Atlas (logs centralizados, middleware de trazabilidad, exception handler global)
- **Integraciones:** PayPal Checkout, LiveKit (videollamadas), RDS PostgreSQL
- **DevOps:** Docker (.dockerignore optimizado), Kubernetes, GitHub Actions (CI/CD), AWS ECR
- **Desafíos Resueltos:** TLS en PHP, Network Access Atlas, Docker build optimization, idempotent entrypoint
- **Repositorios:** [Backend](https://github.com/JoelLeyes/Proyecto2026-Backend_PHP) | [Frontend](https://github.com/JoelLeyes/Proyecto2026-Frontend_PHP) | [Demo](https://app.agendaonline.cloud-ip.cc)
