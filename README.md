## 🚀 Proyectos Destacados

### 📋 [Sistema de Registro de Usuarios en AWS](proyecto-registro.html)
Arquitectura de microservicios en Amazon EKS con:
- **Stack:** Python, Django, PostgreSQL (RDS), Flask, Nginx
- **Infrastructure:** EKS, ECR, ALB, NLB, API Gateway, KubernetesSecrets
- **Security:** SSDLC (Bandit, pip-audit, OWASP ZAP, ECR scanning)
- **Repos:** [Backend](https://github.com/JoelLeyes/BackEnd-proyecto_registro_usuarios) | [Frontend](https://github.com/JoelLeyes/FrontEnd-proyecto_registro_usuarios) | [Notificaciones](https://github.com/JoelLeyes/Notificaciones-proyecto_registro_usuarios)

### 📅 [AgendaOnline: Marketplace SaaS con Observabilidad](proyecto-agenda.html)
Plataforma de servicios profesionales en Amazon EKS con logging centralizado:
- **Stack:** Laravel 10, Vue 3, Vite, PostgreSQL (RDS), MongoDB Atlas
- **Features:** REST API con Sanctum, Pagos (PayPal), Videollamadas (LiveKit), PWA
- **DevOps:** Docker (.dockerignore optimizado), Kubernetes, GitHub Actions (CI/CD), AWS ECR
- **Observabilidad:** Atlas logging con middleware trazabilidad, exception handler global, mongosh queries
- **Repos:** [Backend](https://github.com/JoelLeyes/Proyecto2026-Backend_PHP) | [Frontend](https://github.com/JoelLeyes/Proyecto2026-Frontend_PHP) | [Demo Live](https://app.agendaonline.cloud-ip.cc)
