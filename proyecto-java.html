Chat Empresarial JavaEE — Java EE | Servlets | Nginx | Docker | AWS ECR | EC2 | RDS PostgreSQL | GitHub Actions

Solución web de mensajería interna orientada a empresas, diseñada para centralizar conversaciones, acelerar la coordinación entre equipos y dejar trazabilidad operativa. El proyecto integra un backend Java EE con arquitectura en capas, frontend servido por Nginx y despliegue contenerizado en AWS. Incluye autenticación, chats privados y grupales, reacciones, estado de escritura, notificaciones y un panel de administración, con foco en rendimiento, mantenibilidad y facilidad de despliegue.

Qué resuelve

Está pensado como un chat corporativo para organizaciones que necesitan comunicación interna segura, ordenada y extensible. La propuesta combina experiencia de usuario simple con una base técnica sólida: separación entre capa web, servicios y acceso a datos, además de un flujo de despliegue automatizable.

Arquitectura y alcance

  • Backend: Java EE con Servlets, capa DAO, capa Service y modelos de dominio
  • Frontend: HTML, CSS y JavaScript servidos por Nginx
  • Mensajería: chats privados, grupos, historial de mensajes, reacciones y typing indicator
  • Operación: notificaciones por email y push para eventos relevantes
  • Despliegue: imágenes en ECR, ejecución en EC2 y actualización con GitHub Actions

Habilidades demostradas

  • Arquitectura en capas: separación clara entre DAO, Service, Servlet y modelos
  • Diseño de dominio: entidades para usuarios, chats, mensajes, reacciones, estados y roles
  • Experiencia de producto: panel de administración, login, conversaciones y feedback en tiempo real
  • Automatización: contenedorización con Docker y despliegue repetible en AWS
  • Operación cloud: ECR, EC2, SG, SSH y actualización remota por pipeline

Tecnologías y servicios

  • Backend: Java EE, Servlets, patrón DAO/Service, autenticación y control de sesiones
  • Frontend: HTML5, CSS3, JavaScript, Nginx
  • DevOps: Docker, Docker Compose, GitHub Actions
  • AWS: ECR, EC2, Security Groups, despliegue remoto por SSH
  • Integraciones: email, notificaciones push, chat grupal e indicadores de actividad
