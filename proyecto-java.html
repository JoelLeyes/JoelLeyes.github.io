Frontend JavaEE — Nginx + HTML5 + CSS + JS (Integración con backend Java)

Interfaz de usuario para el proyecto JavaEE: páginas estáticas y templates servidos por Nginx que consumen APIs REST del backend Java. Implementa login, chat básico, panel de administración y estilos responsivos. El frontend está preparado para ser desplegado como contenedor Docker junto con el backend en un entorno orquestado (Kubernetes / Docker Compose).

Contenido

  • Archivos principales: `index.html`, `admin.html`, `login.html`, `styles.css`, `theme.js`
  • Servidor recomendado: Nginx para servir estáticos y proxy a backend Java
  • Contenerización: `Dockerfile` incluido para build y despliegue rápido
