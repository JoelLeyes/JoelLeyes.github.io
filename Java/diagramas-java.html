Notas sobre los Diagramas

  • Todos los diagramas representan el estado actual de la arquitectura en producción.
  • La infraestructura está desplegada en AWS región us-east-1 con certificado HTTPS de ZeroSSL.
  • El pipeline CI/CD está configurado para despliegue automático en cada push a GitHub.
  • Los datos sensibles están en subnets privadas y no son accesibles directamente desde internet.
