Laravel 10 | Vue 3 | WebSockets | Redis | EKS | RDS PostgreSQL

AgendaOnline

Marketplace de servicios con tiempo real, colas y despliegue en AWS

Plataforma de servicios profesionales desarrollada con Laravel 10 como backend API y Vue 3 + Vite como SPA. La arquitectura actual corre sobre Amazon EKS con NGINX Ingress y expone frontend, backend, Reverb, notificaciones y queue-worker dentro del namespace proyecto2026. La sincronización en vivo se resuelve con WebSockets mediante Reverb + Echo, mientras que Redis sostiene cache y colas.

Laravel 10 Vue 3 WebSockets Reverb Echo Redis EKS RDS PostgreSQL MongoDB Atlas PayPal LiveKit

Qué cubre la app

  • Reservas, perfiles de profesionales y gestión de servicios.
  • Pago online con PayPal y videollamadas con LiveKit.
  • Autenticación con Sanctum y panel de administración.

Tiempo real y colas

  • WebSockets con Reverb + Echo para notificaciones y cambios de estado.
  • Redis como cache, cola de jobs y soporte para el queue-worker.
  • Microservicio de notificaciones desacoplado para emails y alertas.

Repositorios del Proyecto

  • Backend (Laravel + Realtime + Jobs)
  • Frontend (Vue 3 + Vite + Echo)
  • Notificaciones (microservicio de email)
  • Demo en Vivo

Arquitectura actual

El flujo real del sistema es Usuario → ALB → nodos EKS → Ingress NGINX → services internos. El frontend, backend, Reverb, Redis, queue-worker y notificaciones viven dentro del namespace proyecto2026. El backend se apoya en RDS PostgreSQL para datos operacionales, MongoDB Atlas para logs, y servicios externos como PayPal, LiveKit y Google OAuth.

Tecnologías y Servicios

Backend

  • Laravel 10, REST API y Eloquent ORM.
  • Sanctum para autenticación y control de sesión.
  • Jobs, eventos y servicios de dominio separados.

Frontend

  • Vue 3, Vite y Pinia.
  • Echo para sincronización UI en vivo.
  • SPA lista para flujos de reservas y notificaciones.

Tiempo real

  • Reverb como servidor de WebSockets.
  • Echo como cliente para canales privados.
  • Redis y queue-worker para procesos asincrónicos.

Infraestructura

  • Amazon EKS, ALB, NGINX Ingress y ECR.
  • Servicios Kubernetes: frontend, backend, reverb, redis y notificaciones.
  • Despliegue con namespaces y secrets en Kubernetes.

Datos

  • RDS PostgreSQL para la base principal.
  • MongoDB Atlas para logs y trazabilidad.
  • Redis para cache y colas.

Integraciones

  • PayPal para pagos online.
  • LiveKit para videollamadas.
  • Google OAuth para login social.

DevOps

  • Docker, kubectl y AWS CLI.
  • GitHub Actions para build, push y redeploy.
  • Scripts PowerShell para automatizar recuperación y despliegue.

Observabilidad

  • MongoDB Atlas como registro centralizado.
  • Trazabilidad de accesos, errores y eventos del sistema.
  • Separación clara entre datos operacionales y logs.

Características Técnicas Destacadas

  • Tiempo real con WebSockets: Reverb + Echo sincronizan reservas, notificaciones y estados de la interfaz sin recargar la página.
  • Notificaciones desacopladas: El servicio de notificaciones envía emails y alertas desde un microservicio separado.
  • Procesamiento asincrónico: Redis y queue-worker resuelven jobs, correos y tareas diferidas.
  • Arquitectura en EKS: ALB, Ingress NGINX y Kubernetes coordinan frontend, backend y servicios auxiliares.
  • Integración multi-servicio: PayPal, LiveKit y Google OAuth amplían el alcance de la app sin mezclar responsabilidades.
  • Persistencia separada: PostgreSQL guarda la operación de negocio, mientras MongoDB Atlas conserva logs y trazabilidad.
  • CI/CD repetible: GitHub Actions construye imágenes Docker, publica en ECR y actualiza el clúster.

Despliegue y CI/CD

  • El backend, frontend y notificaciones se empujan como imágenes Docker a ECR.
  • Los workflows actualizan kubeconfig, aplican manifests y reinician deployments en Kubernetes.
  • El tráfico público entra por ALB y se enruta internamente con NGINX Ingress.
  • Redis, Reverb y queue-worker forman la capa de soporte para realtime y procesos diferidos.
  • Los secrets de Kubernetes concentran credenciales de RDS, PayPal, LiveKit, Google y el servicio de notificaciones.
